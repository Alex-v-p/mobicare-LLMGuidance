# Drug-dosing pipeline

This project now includes a separate `drug_dosing` pipeline variant.

It is intentionally deterministic and does **not** call the generative answer path.

Current scope:
- MRA (spironolactone/eplerenone)
- Beta-blocker
- RAS (ACE-I style titration path)
- ARNI (sacubitril/valsartan)
- SGLT2 inhibitor
- Loop diuretic
